Atoms — Buttons

shadcn-inspired variant system. Outfit font, medium weight. Brand gradient reserved for primary CTA.

Variants (Medium Size)
Default
Primary. midnight-900 fill, white text. The default action button.
Secondary
Muted fill. For secondary actions alongside a primary.
Outline
Border only. For tertiary actions, filters.
Ghost
No border, no fill. Hover reveals bg. For dismiss, back, low-priority.
Link
Underlined text. No padding. For inline navigation.
Destructive
Red fill. Reserved for delete/remove actions.

grad-cool. The ONE gradient button. Only for the main CTA on cards and hero areas.
